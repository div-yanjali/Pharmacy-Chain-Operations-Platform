System Architecture
Python stdlib microservices · SQLite/PostgreSQL · REST APIs · JWT Auth · Event-driven design
Architecture Principles
Domain-driven microservices · Event-driven via Kafka for async workflows · CQRS for read/write separation · Offline-first sync (SQLite → PostgreSQL) for intermittent connectivity · Row-level security for multi-store isolation · Immutable audit log via append-only partitioned tables · JWT RS256 + refresh token rotation · Redis cache for sub-50ms inventory reads during peak hours

Non-Functional Design
⚡ Performance
Redis cache for inventory reads (p99 <50ms). Materialized view refresh for BI. PgBouncer connection pooling. K8s HPA autoscaling. Rate limiting per store.
🔒 Security
JWT RS256 + refresh rotation. PostgreSQL RLS. AES-256 at rest. TLS 1.3 in transit. OWASP Top 10. HashiCorp Vault secrets. Regular pen-testing.
📡 Offline Resilience
Local SQLite cache for semi-urban stores. Offline sale queue. CRDT merge on reconnect. Outbox pattern. 72hr offline window.
📊 Observability
Prometheus + Grafana dashboards. OpenTelemetry + Jaeger tracing. Structured JSON logging to ELK. PagerDuty SLO alerts.
🔄 Event Architecture
Kafka topics: stock.updated, sale.completed, anomaly.detected. Dead-letter queues. Avro schema registry. Consumer groups per domain.
📋 Compliance
DPDP Act 2023. CDSCO pharmacy automation guidelines. Immutable Schedule H audit. GSTIN invoicing. 7-year data retention.
